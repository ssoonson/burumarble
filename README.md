# 🎲 부루마블 (Buru-Marble)

초등학생들과 태블릿으로 함께 즐기는 보드 게임 웹 앱입니다.

## 🌐 플레이하기

**GitHub Pages에서 바로 실행:**

👉 **https://ssoonson.github.io/burumarble/**

브라우저(Chrome, Safari 등)에서 위 주소를 열면 별도 설치 없이 플레이할 수 있습니다.

## 🎮 게임 방법

1. 플레이어 수(2~4명)와 동물 이모지를 선택합니다.
2. **주사위 굴리기**로 보드판을 시계방향으로 이동합니다.
3. 빈 땅을 사고, 건물을 업그레이드하세요.
4. 상대 땅에 걸리면 통행료를 냅니다.
5. 출발지(0번)를 지날 때마다 월급 20만 원을 받습니다.

## 🛠️ 로컬 실행

```bash
# 저장소 클론 후 index.html을 브라우저에서 열기
open index.html
```

## 📁 구성

- `index.html` — HTML, CSS, JavaScript 단일 파일 게임
